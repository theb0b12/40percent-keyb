
# 40% keyb

A minivan project that I completed in about 3 hours.

Made with ergogen.

zmk repo is https://github.com/theb0b12/zmk-config-fourty

INFO:
- 45 keys
- 12.75u 40% keyb
- Arrows version, with the 3 keys in the bottom-right are replaced with 1.5/1/1/1u to allow for a traditional orthogonal arrow key arrangement
